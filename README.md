# Sistema de Pedidos - Ramen & Gyozas

Este proyecto es un sistema de gestión de pedidos desarrollado para un restaurante de ramen y gyozas.

## Tecnologías utilizadas
- HTML
- CSS
- JavaScript
- Node.js
- Express
- SQLite
- GitHub

## Funciones del sistema
- Registrar pedidos
- Calcular total de ventas
- Guardar ventas en base de datos
- Consultar ventas registradas

## Instrucciones para ejecutar el proyecto

1. Abrir el proyecto en Visual Studio Code
2. Abrir la terminal
3. Ejecutar el servidor con el comando:

node backend/app.js

4. Abrir el archivo en el navegador:

frontend/pedidos.html

5. Registrar pedidos y calcular total
